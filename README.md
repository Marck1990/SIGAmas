SIGA+

SIGA+ es una aplicación web orientada al monitoreo y visualización de información agroclimática, diseñada para facilitar el análisis de condiciones ambientales mediante herramientas visuales e interactivas.

Funcionalidades

Visualización de mapas interactivos mediante Leaflet.
Búsqueda dinámica de localidades.
Renderizado de polígonos geográficos según zonas climáticas.
Cambio de color automático de áreas según estado climático.
Popup informativo por región/localidad.
Interfaz responsive y adaptable.
Tecnologías Utilizadas

HTML5
CSS3
JavaScript
Leaflet.js
APIs de datos climáticos / geográficos
Objetivo del Proyecto

El objetivo de SIGA+ es brindar una herramienta visual e intuitiva que permita representar información agroclimática de forma clara, facilitando la toma de decisiones en contextos productivos y ambientales.

Aprendizajes Aplicados

Durante el desarrollo de este proyecto se trabajó sobre:

Manipulación avanzada del DOM.
Integración con librerías externas.
Consumo e interpretación de datos externos.
Renderizado dinámico de mapas y elementos visuales.
Organización modular de código JavaScript.
